 Penjelasan : 
 Variabel adalah tempat untuk menyimpan data. Di Kotlin ada dua jenis:
 1. 'val' (Value) : Nilainya dikunci/permanen, tidak bisa diubah setelah diisi.
 2. 'var' (Variable): Nilainya dinamis, bisa diubah-ubah seiring berjalannya program.

 Penjelasan :
 Fungsi (fun) digunakan untuk membungkus blok kode agar bisa digunakan berulang kali
 tanpa harus menulis ulang kodenya. Fungsi bisa menerima data masukan (disebut Parameter).
 Membuat fungsi bernama 'hitungTotalBayar'
 Fungsi ini meminta dua data masukan (parameter): harga (Int) dan jumlah (Int). ': Int' di akhir kurung menandakan bahwa fungsi ini akan menghasilkan/mengembalikan nilai berupa angka bulat.
